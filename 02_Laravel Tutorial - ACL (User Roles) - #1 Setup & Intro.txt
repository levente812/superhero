   php artisan db:seed
   
   hiba:
    Class DatabaseSeeder does not exist
   Megoldás:
    Nem elég a seeder fileokat bemásolni, azokat artisannal kell létrehozni, 
       hogy szeresse.
    php artisan make:seeder UsersTableSeeder
    php artisan make:seeder RolesTableSeeder
   // létre jön 2db file, amibe bemásolom a tartalmat
   // import classokat intézem:
		use App\User;
		use App\Role;

    Ezután már fut:
      php artisan db:seed
